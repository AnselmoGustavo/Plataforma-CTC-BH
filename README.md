# CTCBH - Sistema de Gestão de Espaços e Vagas

Projeto acadêmico desenvolvido para o curso de Sistemas de Informação (2º semestre 2025). O sistema permite gerenciar
aluguéis de vagas de estacionamento e de salas/espaços, controle de parceiros, eventos, voluntários e geração de relatórios em PDF.

Principais funcionalidades:
- Cadastro e controle de vagas de estacionamento
- Cadastro e controle de salas/espaços (nome, responsável, finalidade, descrição)
- Filtros e busca unificada (vagas e salas)
- Geração de relatórios em PDF com cabeçalho padrão
- Gestão de eventos, voluntários e relatórios financeiros (módulos existentes no projeto)

## Tecnologias
- Frontend: React 18, TypeScript, Vite, TailwindCSS
- Backend: .NET 8.0 (ASP.NET Core), Entity Framework Core
- Banco de dados: SQL Server (local) / EF Migrations
- Bibliotecas: Axios, jsPDF, autoTable, shadcn/ui, lucide-react
